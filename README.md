# Telega - Мессенджер

## Структура проекта

```
telega-clone/
├── server.py         # Python сервер (API)
├── index.html        #Frontend
├── data/
│   ├── users.json    # База пользователей
│   └── messages.json # База сообщений
└── README.md        # Этот файл
```

## Запуск

```bash
# Запуск сервера
python server.py

# Открыть в браузере
http://<твой IP>:8080
```

## API Endpoints

- `GET /api/users?username=xxx` - Список пользователей
- `POST /api/login` - Вход {username: "xxx"}
- `POST /api/send` - Отправить сообщение {sender: "xxx", recipient: "yyy", text: "сообщение"}
