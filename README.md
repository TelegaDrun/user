# TelegaDrun (GitHub Pages)

Статическая версия мессенджера для GitHub Pages.

## Как это работает

- Приложение полностью фронтендное: `index.html` + `gh-pages-api.js`.
- Запросы к `/api/*` обрабатываются прямо в браузере.
- Все данные (пользователи, друзья, сообщения, группы, настройки) хранятся в `localStorage`.
- Локальный Python/Node сервер больше не нужен.

## Деплой на GitHub Pages

1. Запушь репозиторий на GitHub.
2. Открой **Settings -> Pages**.
3. В разделе **Build and deployment** выбери:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` (или твоя ветка), папка `/ (root)`
4. Сохрани и дождись публикации.

## Примечания

- Данные живут только в браузере пользователя и не синхронизируются между устройствами.
- Чтобы сбросить базу, очисти `localStorage` для сайта.
