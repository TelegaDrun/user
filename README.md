Коротко: это минимальный статический сайт для бесплатного хостинга.

Как разместить
- Netlify: перетащите папку проекта в дашборд Netlify (drag & drop) или подключите репозиторий.
- Vercel: установите vercel CLI и выполните команду vercel из папки проекта.
- Surge: npm i -g surge; surge .
- GitLab Pages: разместите файлы в репозитории GitLab и активируйте GitLab Pages.

Файлы в этом наборе:
- telegadrun2/index.html — основная страница
- telegadrun2/style.css — стили
- telegadrun2/script.js — минимальный скрипт

Если нужно, добавлю конфиги под конкретную платформу.
