GitHub Pages — минимальный пример

Как использовать
- Поместите содержимое папки telegadrun2/github-pages в папку docs или корень вашего репозитория, в зависимости от настроек Pages.
- В настройках репозитория GitHub Pages выберите источник: /docs или ветку, где располагаются файлы.
- Откройте страницу https://<ваш_пользователь>.github.io/<ваш_репозиторий>/ для проверки.

Советы
- При использовании SPA учтите, что все маршруты могут не работать без поддержки 404-redirect. Здесь добавлен 404.html, который перенаправляет на index.html.
- В случае статического сайта без роутинга — всё будет работать из коробки.
